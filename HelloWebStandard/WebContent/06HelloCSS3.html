외부 스타일시트 : 모든 문서를 대상으로 공통으로 적용할 스타일을 정의

CSS Reset

CSS를 적용하더라도 모든 브라우저에 동일하게 보이는 것은 아님

각 브라우저마다 기본적으로 적용되어 있는 스타일의 차이 때문인데

이것을 CSS Reset이라는 특수한 기능으로 모든 브라우저의 기본 스타일(여백등등)을 초기화0 시킬수 있음

normalize.css 를 이요해서 CSS 적용을 표준화 할 수 있음
